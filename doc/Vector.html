Class: Vector

Vector()

The Vector class represents an "n" dimensional vector Dependecies: - misc.js - matrix.js (cross product)

Constructor

new Vector()

Creates a Vector. The arguments can be: - Nothing: The Vector will be 0 dimensional - One Vector: In this case this is a copy constructor - A random object (except a "Number"): In this case searches for x,y,z coordinates, it's dimension becomes the highest possible and inherits the x,y,z coordinates - arguments[0] == 'init' and arguments[1] a number which will be the vector's dimensions (every dimension's vaule is arguments[2] or 0 if omitted) - One number parameter for every desired dimension. The given numbers will be the dimensions values - An array of the above 2 options - A JSON string of the above 4 options works the same
